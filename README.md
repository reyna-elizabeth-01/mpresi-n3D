## Proyectos destacados 🚀

### 🔹 Automatización de Notificaciones con n8n
**Descripción:** Diseño e implementación de un ecosistema automatizado que extrae datos de estudiantes desde Google Sheets, procesa información mediante HTTP Requests y distribuye correos personalizados. El proyecto destaca por la resolución de errores de validación de datos mediante la implementación de nodos **Filter** y lógica **Switch** para asegurar la integridad del envío.

### 🔹 Fabricación Digital e Impresión 3D
**Descripción:** Flujo de trabajo profesional de manufactura aditiva: desde el bocetaje y modelado 3D en CAD, hasta el laminado (Slicing) con **Cura** y la impresión final en equipos **Ender-5 Pro**.
